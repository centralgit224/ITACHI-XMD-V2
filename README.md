## 🚀 Installation

### 1. Cloner le projet
```bash
git clone https://github.com/centralbot224/ITACHI-XMD
cd ITACHI-XMD-main
```

### 2. Installer les dépendances
```bash
npm install
```

### 3. Configurer le bot
```bash
cp .env.example .env
```

Ouvrez le fichier `.env` et remplissez :
```env
OWNER_NUMBER=224XXXXXXXXX
OWNER_NAME=Votre Nom
BOT_NAME=ITACHI-XMD
PREFIX=.
COMMAND_MODE=public
```

### 4. Démarrer le bot
```bash
npm start
```
